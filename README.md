# PNG to PNM Converter

This C program converts PNG files to PNM (Portable Any Map) format. It utilizes zlib for decompressing PNG data.

## Overview

The PNG to PNM Converter is a command-line tool designed to convert PNG images to the PNM format. It leverages zlib, a
widely-used compression library, to decompress the PNG data and extract the image information.

## Usage

The paths to input and output files are passed as first and second arguments.
